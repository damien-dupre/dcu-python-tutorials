# What is Python?

Python is one of the most popular programming languages in the world. It is a general-purpose language used for web development, automation, scientific computing, data analysis, and much more.

Python is known for being **readable** and **beginner-friendly**. Its syntax is clean and emphasises code readability.

For data analysis, Python is usually used via environments like **Google Colab**, **Jupyter Notebooks**, or **VS Code**.

---

# How do I code in Python?

While it is not required to be an expert programmer to use Python, there is still a set of basic concepts that new users need to understand.

To help you learn, we will always use a different font to distinguish regular text from `computer_code`. The best way to master these topics is through practice and lots of repetition.

One thing that intimidates new Python users is how it reports **errors**. Python reports errors in red text, which makes it seem like it is scolding you. However, seeing red text is not always bad.

- If the text says **"Error"** or **"Exception"**, figure out what is causing it. Something is wrong!
- If the text says **"Warning"**, figure out if it is something to worry about. Everything is working, but pay attention.
- Otherwise, the text is just output. Read it and carry on.

---

# Useful tips to keep in mind

1. **Computers are not actually that smart**: Your instructions can't have mistakes or ambiguity.

2. **Take the "copy, paste, and tweak" approach**: It is often easier to modify existing code than to write from scratch.

3. **The best way to learn to code is by doing**: Learning goes much smoother when you have a goal in mind or are working on a project you care about.

4. **Practice is key**: The only way to improve your coding skills is through lots of practice.
